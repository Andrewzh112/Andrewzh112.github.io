# Plots Directory

This directory contains PDF plots generated from the CSV data.

## Generating Plots

To generate plots, you need Python with pandas and matplotlib:

```bash
pip install pandas matplotlib

# From the on-policy-distillation-adventures directory
python scripts/generate_plots.py
```

## Expected Files

The blog expects these PDF files:

1. `mixing_comparison.pdf` - Comparison of mixing strategies
2. `oracle_scaling.pdf` - Teacher size scaling analysis
3. `reward_comparison.pdf` - Different reward functions
4. `stability_analysis.pdf` - Training stability metrics
5. `seq_vs_kl.pdf` - Sequential vs standard KL updates
